# Fitbit-smartwatch-analysis-dashboard
Power BI dashboard and statistical analysis exploring user perceptions and future usage intention towards Fitbit smartwatches.
## Overview
This project analyzes user perceptions toward Fitbit smartwatches using survey-based quantitative and qualitative analysis.

## Tools Used
- Power BI
- Excel
- Correlation Analysis
- Regression Analysis

## Key Findings
- Ease of use emerged as the strongest predictor of future usage intention.
- Fitbit Premium adoption remained limited among respondents.
- Pricing, tracking accuracy, and feature value were common improvement areas.

## Dashboard Pages
1. Executive Overview  
2. User Perception Analysis  
3. Fitbit Premium Insights  
4. Qualitative Insights & Recommendations
